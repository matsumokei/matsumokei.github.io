# ブログへようこそ

このブログは**Next.js**と**Markdown**を使って構築されています。

## 特徴

- 📝 Markdownで記事を書ける
- 🏷️ タグ機能
- 🎨 Tailwind CSSでスタイリング
- ⚡ 高速なビルド

## コード例

```typescript
const greeting = (name: string) => {
  return `Hello, ${name}!`;
};

console.log(greeting("World"));
```
